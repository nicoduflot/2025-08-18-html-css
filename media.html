Les images

Les images peuvent être affichées à l'aide de deux balises : img et picture

img

img est une balise qui une fois écrite, va aller chercher l'image source et l'afficher à sa place. Si rien d'autre n'est précisé, l'image s'affiche à sa taille originale. Si on change, par exemple, sa largeur, la hauteur qui le ratio (l'homothétie, le rapport hauter / largeur)

Une image peut être appelé dans un paragraphe : ou hors paragraphe.

Image hors paragraphe :

La balise figure

Il existe une balise conteneur pour les media, qui aide la sémantique web pour le référencement et l'accessibilité. la balise figure va encadrer un media, et donne la possibilité d'y ajouter une légende, avec la balise figcaption.

La légende sera rattachée sémantiquement à l'image et son texte alternatif, ce qui améliorera le référencement.

L'audio

Pour mettre à disposition un fichier audio sur un site, on utlise la balise audio. Cette balise peut soit appeler un seul fichier, ou utiliser aussi la balise source. Utiliser plusieurs sources permet de s'adapter à la configuration de l'utilisateur. Un appareil, pour lire un fichier audio (ou vidéo) doit posséder les bon codecs. Si je met a disposition un fichier avec une extension que l'appareil ne connaît pas, j'aurais une erreur. Si je propose le même son dans des formats différent avec source, l'appareil lancera le premier fichier qu'il est capable de lire.

La balise video

La balise vidéo se comporte comme la balise audio.

L'iframe

L'élément iframe permet d'inserrer une page internet dans le document, mais pas que, il est possible d'inserrer d'autre type de fichier, par exemple des pdf ou directement des fichier vidéo, sons, etc.

L'iframe permet aussi d'afficher des ressources à l'intérieur depuis la page qui la contient
